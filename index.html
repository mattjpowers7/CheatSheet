HTML Tag Family Tree

Grandparents -> Grandchildren

Tag Name Description
<html> HTML The html tag comes after the <!DOCTYPE html> declaration, but otherwise begins and ends the html document.
<head> Header Before the body Where the CSS style links go. Don't put content of the webpage here!
<title> Title Put the title of your webpage here and it will appear when you hover over the tab.
<body> Body Put the content of your webpage here.
<h1>, <h2>, etc.. Heading Headings for other content like text blocks, articles, or images.
<table> Table Creates a table where you can put rows, headings, data, columns, etc...
<p> Paragraph Blocks of text, like paragraphs, all in one element.
<div> Dividers Anything can go in a div. Just think of it like a container.
<a> Links You can insert any link using this tag, thus making text and images clickable
<img> Images Use this tag to input images into your webpage, but keep in mind it's a self-closing tag
